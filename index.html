Llamashot

A fast, lightweight screenshot and screen recording tool for Windows. Capture, annotate, record, and extract text.

WPF / .NET 10 Open Source Windows 10/11
Download Installer Explore Features

Everything You Need

Screenshot, annotate, record, and extract text — all in one lightweight tool

Region Capture

Capture full screen, window, or custom region instantly. Drag to select with live dimensions and resize handles.

Screen Recording + Audio

Record any region as MP4 with unlimited duration. Capture microphone audio. Pause and resume anytime.

11 Annotation Tools

Pencil, arrow, rectangle, ellipse, text, marker, blur, stamps, and eraser. Configurable color and thickness.

Text Extraction (OCR)

Extract text from any area of the screen with one click. Auto-inverts dark backgrounds for better recognition.

Screenshot History

Thumbnail gallery with multi-select, bulk copy, and delete. Green “Saved” and blue “Copied” badges.

Quick Sharing

Copy, save, or pin your capture in seconds. Multiple output formats: PNG, JPG, BMP with quality control.

22 Keyboard Shortcuts

Every tool and action has a shortcut. Click-to-record in Settings. Duplicate detection prevents conflicts.

Lightweight & Fast

Tiny memory footprint. Instant startup. No background services. Self-contained single-file executable.
