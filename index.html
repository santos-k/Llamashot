Powerful Features, Zero Bloat

Every feature designed with speed and simplicity in mind

Snipping Toolbar

Snipping Toolbar

Press PrintScreen and a sleek toolbar appears at the top of your dimmed screen — just like Windows Snipping Tool, but more powerful. Switch between Screenshot, Recording, Long Shot, and OCR modes, choose your capture type, set a delay, and go.

  • Four capture modes — Screenshot (1), Recording (2), Long Shot (3), OCR (4)
  • Three capture types — Region (draw), Window (hover & click), Fullscreen (instant)
  • Window capture — hover over any window to highlight with blue border, click to capture
  • Delay timer — no delay, 1s, 3s, 5s, or 10s countdown before capture
  • Keyboard shortcuts — 1/2/3 for modes, R/W/F for capture types
  • Seamless default — start drawing immediately, toolbar auto-hides
New in v6.0

Long Shot — Scrolling Screenshot

Capture entire scrollable pages, chat threads, or long documents in a single image. Select Long Shot mode (3), click any window, and the app scrolls and stitches frames automatically. Or scroll manually and let the app capture as you go.

  • Auto scroll — app scrolls via Page Down and captures frames automatically
  • Manual scroll — scroll yourself, app captures on each scroll event
  • Auto/Manual toggle — persisted preference on toolbar, remembered across sessions
  • Intelligent stitching — pixel-row overlap detection removes duplicates
  • Fixed element detection — headers and footers are captured once, not repeated
  • Full annotation toolkit — all 11 tools available in preview window
  • Handle-based crop — 8 handles for precise cropping before save
  • Stop & Cancel — Stop keeps result, Cancel discards, Esc to stop mid-scroll
Capture

Precision Region Capture

Press PrintScreen and drag to select any area of your screen. The interactive overlay shows a crosshair cursor with live dimension display, so you always know the exact pixel size of your selection.

  • Custom region selection — drag to select any area with live width x height display
  • 8 resize handles — fine-tune your selection from any corner or edge
  • Full-screen toggle — double-click to expand selection to entire screen
  • Multi-monitor support — captures across all connected displays seamlessly
  • Pan mode — hold Space to drag and reposition your selection (Photoshop-style)
  • Capture mouse cursor — optional setting to include cursor in screenshots
Recording

Screen Recording with Audio

Record any region of your screen as high-quality MP4 video. Select and resize your region with the same handles as screenshots, toggle audio before starting, then hit Start with a 3-2-1 countdown.

  • Pre-start flow — resize/move region with handles, toggle mic/sys audio, click Start with 3-2-1 countdown
  • Full annotation toolkit — all 11 screenshot tools available during recording with color picker & thickness
  • MP4 output — auto-selects HD1080p, HD720p, or VGA based on region size
  • Microphone + System audio — independent toggles before and during recording
  • Pause & resume — press Space to pause, press again to continue
  • Tool toggle — tools stay active between strokes, click/shortcut again to deselect
  • Clean saving UI — all tools hidden during save, showing only “Saving…” message
  • Timer display — live elapsed time in the floating toolbar
Annotate

11 Professional Annotation Tools

Mark up your screenshots with a comprehensive set of drawing tools. Each tool is accessible with a single keypress. Choose from 24 preset colors and 10 thickness levels to make your annotations stand out.

  • Pen — freehand drawing with smooth round caps
  • Arrow & Line — straight lines with optional arrowhead
  • Rectangle & Ellipse — shape outlines for highlighting areas
  • Text — add multi-line text labels with customizable font size
  • Marker — semi-transparent highlighter effect
  • Blur / Pixelate — redact sensitive information with mosaic effect
  • Stamps — green checkmark and red cross for approvals/rejections
  • Eraser — click any annotation to remove it individually
  • Undo / Redo — full action history with Ctrl+Z / Ctrl+Y
OCR

Text Extraction (OCR)

Extract text from any area of your screen — select OCR mode in the toolbar, draw a dashed region, and text is instantly copied to your clipboard. Powered by Windows built-in OCR engine with intelligent preprocessing for maximum accuracy.

  • Sub-region selection — drag to select the exact text area after capture
  • Multi-language support — detects your system languages automatically
  • Dark background inversion — auto-detects and inverts dark images for better recognition
  • Contrast enhancement — 1.5x contrast boost for clearer character edges
  • Smart upscaling — scales small text up to 4x for improved accuracy
  • Result window — shows extracted text with word and character count
  • One-click copy — select all and copy buttons for quick use
History

Screenshot History & Management

Every capture is saved to a searchable thumbnail gallery. Quickly find, copy, save, or delete past screenshots. Bulk operations let you manage dozens of captures at once.

  • Thumbnail gallery — visual grid of all captures with preview images
  • Status badges — green "Saved" and blue "Copied" labels on each item
  • Per-item actions — copy, save as, or delete individual captures
  • Bulk selection — select all checkbox and multi-select for batch operations
  • Bulk copy & save — copy multiple to clipboard or save to a folder
  • Metadata display — timestamps, dimensions, file path, and size on hover
  • Click to open — click any thumbnail to open the file directly
  • Configurable limit — set max history items (default: 100)
Output

Pin, Share & Export

After capturing, instantly choose what to do with your screenshot. Pin it as a floating reference window, copy to clipboard, or save in your preferred format. The pin window stays on top of all other windows for easy reference.

  • Pin on screen — floating borderless window positioned at capture location
  • Adjustable opacity — mouse wheel to set 10%–100% transparency
  • Quick copy — double-click pinned image to copy to clipboard
  • Draggable — left-click and drag to reposition anywhere
  • Save dialog — choose format (PNG, JPG, BMP) and location
  • JPEG quality control — slider from 10–100 for fine-tuning file size
  • Auto-naming — screenshots named with timestamp: screenshot_yyyyMMdd_HHmmss
  • Remembers directory — defaults to your last save location

And Much More

Every detail considered for a seamless experience

Multi-Monitor Support

Captures across all connected displays. The selection overlay spans your entire virtual desktop, so you can grab content from any monitor.

Delayed Capture

Set a countdown timer (1s, 3s, 5s, or 10s) before capture starts. Perfect for capturing tooltips, dropdown menus, or context menus.

Fullscreen Shortcuts

Instantly save or copy the entire screen with a single hotkey. Shift+PrtSc saves to file, Ctrl+PrtSc copies to clipboard. No UI, no delay.

Cursor Capture

Optionally include the mouse cursor in your screenshots and recordings. Enable in settings — great for tutorials and documentation.

Seamless Auto Update

Check for updates from the About window. Downloads and installs silently in the background — no dialogs, no interruptions. App restarts automatically with the new version.

Window Capture

Select Window mode in the snipping toolbar, then hover over any window to see it highlighted with a blue border. Click to capture just that window — works for screenshots, video, and OCR.

Color Memory

Your selected annotation color is remembered across sessions. Pick a color once and it stays — no need to re-select every time you capture.

System Tray Integration

Lives in your system tray. Right-click for quick access to all capture modes, history, settings, and more. Left-click to start a capture. Record Screen opens the snipping toolbar with video mode pre-selected.

Single Instance

Enforces only one running instance via mutex. Start with --silent flag for auto-launch at Windows startup without any visible UI.

Crash Logging

Unhandled exceptions are logged to %APPDATA%/Llamashot/crash.log with timestamps and full stack traces. App continues running after logging.

Recording Exclusion

The recording toolbar and border are automatically excluded from the capture using WDA_EXCLUDEFROMCAPTURE, so your controls never appear in the video.
